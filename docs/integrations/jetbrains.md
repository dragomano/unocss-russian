# Плагин для IDE JetBrains {#jetbrains-ides-plugin}

[Плагин](https://github.com/re-ovo/unocss-intellij) от сообщества, обеспечивающий поддержку UnoCSS в IDE от JetBrains.

:::info
Примечание: Это **плагин, поддерживаемый сообществом; он не проверяется и не поддерживается официальной командой UnoCSS**. Пожалуйста, самостоятельно оцените риски перед использованием. Если у вас возникнут проблемы с этим плагином, пожалуйста, сообщите о них в [re-ovo/unocss-intellij](https://github.com/re-ovo/unocss-intellij).
:::

## Установка {#installation}

[JetBrains Marketplace](https://plugins.jetbrains.com/plugin/22204-unocss)

## Возможности {#features}

- Автодополнение для классов UnoCSS
- Всплывающая документация для классов UnoCSS
- Сворачивание кода
- Предпросмотр цветов/иконок

## Сообщения об ошибках / Запросы функций {#bug-reports-feature-requests}

Сообщайте о них в [Issue Tracker](https://github.com/re-ovo/unocss-intellij/issues)
